# Vuex-learning-demo

> Little demo developed by Vue.js and Vuex.

## Overview

![vuex](https://raw.githubusercontent.com/vuejs/vuex/master/docs/en/vuex.png)

### Usage

```bash
// clone project
git clone git@github.com:AlexBai1991/learning-vuex.git vuex-pro

cd vuex-pro && npm install

npm start
```

### Technology

- gulp
- webpack
- ES2015
- Vue
- Vuex
- Sass
